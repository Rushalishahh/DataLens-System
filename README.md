# DataLens System

This project compares baseline and current datasets, performs EDA, preprocessing, and shows changes, insights, and visualizations.

## Features

- Compare baseline and current datasets  
- Automated EDA: summary statistics, correlations, distributions, missing values  
- Track preprocessing steps applied to data  
- Visualizations: distributions, missing values, and anomalies  
- Flask API for easy integration  

## Tech Stack

- Python  
- Flask  
- Pandas, NumPy  
- Matplotlib, Seaborn, Plotly  
- Scikit-learn
